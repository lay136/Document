知识点：
v-bind 可以动态地绑定一个或多个特性，或一个组件 prop 到表达式
<!-- v-bind: 全称写法 -->
<!-- <a v-bind:href="Url">淘宝3</a> -->
<!-- v-bind: 缩略写法 -->
<a :href="Url">淘宝4</a>

v-on:click 可以缩写为 @
例子：
<button v-on:click="count++">1你点击了{{count}}次</button>
<button @click="count++">2你点击了{{count}}次</button>


修饰符：
.stop  阻止冒泡
.prevent 阻止默认行为
.capture 在捕获阶段执行
.self 只有在自身触发的事件才会执行,不会隔断冒泡
.once 自身事件只触发一次,冒泡不会停止
